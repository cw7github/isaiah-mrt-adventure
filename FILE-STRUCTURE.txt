TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════

Problem: Shapes not draggable
    → Check file loading order (see above)
    → Check browser console for errors (F12)
    → Verify files are in correct directory

Problem: Visuals look the same
    → Clear browser cache (Ctrl+Shift+R)
    → Verify manim-3d-enhancements.css is loading
    → Check CSS file path is correct

Problem: Touch not working
    → Add viewport meta tag (see demo HTML)
    → Test on actual device (not just browser emulation)
    → Check console for errors
