# Ghibli-Style Background Scene Generation Guide

## Overview
This guide provides detailed prompts for generating 24 unique Miyazaki/Ghibli-style background images for Isaiah's MRT Food Adventure reading app.

## Generation Methods

### Option 1: Automated Generation (Experimental)
Run the Node.js script:
```bash
node generate_scenes.js
```

This will attempt to generate images via OpenRouter API. However, free models may not support image generation.

### Option 2: Manual Generation (Recommended)
Use these prompts with AI image generators like:
- **Midjourney** (Discord bot, subscription required)
- **DALL-E 3** (via ChatGPT Plus or API)
- **Stable Diffusion** (local or via services like DreamStudio)
- **Bing Image Creator** (free, powered by DALL-E)

## Style Guidelines
All images should follow these specifications:

- **Art Style**: Studio Ghibli/Hayao Miyazaki watercolor illustration
- **Color Palette**: Soft pastel colors, warm lighting
- **Mood**: Whimsical, magical, calming, child-friendly
- **Orientation**: Horizontal/landscape (ideal for background use)
- **Important**: NO text in images, NO people/characters
- **Resolution**: 1920x1080 or higher recommended

## Scene Categories

Each station has 3 scene types:

1. **Arrival** - The entrance/exterior view when first arriving at the station
2. **Exploring** - Inside view showing the food displays and environment
3. **Enjoying** - Close-up of food service area or eating space

## Tips for Best Results

### For Midjourney:
- Add `--ar 16:9` for landscape orientation
- Add `--v 6` for latest version
- Add `--style raw` for more control

### For DALL-E 3:
- Emphasize "horizontal landscape" in prompt
- Specify "1920x1080 resolution"
- Can use ChatGPT Plus interface for easier generation

### For Stable Diffusion:
- Use models trained on anime/Ghibli style
- Set resolution to 1920x1080 or 1728x1080
- Use negative prompts: "text, letters, words, people, characters, ugly, blurry"

## Quality Checklist

Before using images, verify:
- [ ] No text or words appear in the image
- [ ] No people or characters visible
- [ ] Horizontal/landscape orientation
- [ ] Soft, calming colors appropriate for children
- [ ] Whimsical, magical Ghibli-style aesthetic
- [ ] Clear and not too busy (won't distract from reading text)
- [ ] Appropriate resolution (minimum 1280x720, ideally 1920x1080)
