## External Dependencies

The app uses only **one external dependency**:

- **Google Fonts** (Cinzel Decorative & Lora)
  - Loaded from: `https://fonts.googleapis.com`
  - Required for proper typography
  - Falls back to system serif fonts if unavailable

**Everything else is self-contained!**

## Support & Help

### If Something Goes Wrong
1. Check [TROUBLESHOOTING.md](TROUBLESHOOTING.md)
2. Open browser console (F12) and check for errors
3. Try `test.html` to isolate the issue
4. Verify all files are in the same directory

### If You Want to Customize
1. **Stories**: Edit `pilot-story.js`
2. **Colors**: Edit CSS variables in `styles.css`
3. **Creatures**: Edit zone arrays in `app.js`
4. **Speed**: Edit animation durations in `app.js`

### If You Want to Learn More
1. Read [PROJECT_SUMMARY.md](PROJECT_SUMMARY.md) for complete documentation
2. Review [FEATURES.md](FEATURES.md) for implementation details
3. Explore the code files with comments
